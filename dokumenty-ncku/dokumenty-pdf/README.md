## 📥 Jak pobrać dokumenty jako PDF?

### Metoda 1: Bezpośredni wydruk z przeglądarki
1. Otwórz plik `.html` w przeglądarce (dwukrotne kliknięcie)
2. Kliknij przycisk **"Drukuj / Zapisz jako PDF"** na stronie
3. Lub naciśnij `Ctrl+P` (Windows) / `Cmd+P` (Mac)
4. W oknie drukowania wybierz **"Zapisz jako PDF"** lub **"Microsoft Print to PDF"**
5. Kliknij **"Zapisz"** i wybierz lokalizację

### Metoda 2: Dla zaawansowanych użytkowników
Możesz użyć narzędzi konwersji HTML→PDF:
- **wkhtmltopdf**: `wkhtmltopdf plik.html plik.pdf`
- **Pandoc**: `pandoc -s plik.html -o plik.pdf`
- **Chrome headless**: `chrome --headless --print-to-pdf=plik.pdf plik.html`
